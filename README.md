# 🌌 COSMOS — A Living Universe Portfolio

> A full-screen interactive universe where projects are planets and anyone in the world can add their star.

🌐 **Live →** https://cosmo-steel.vercel.app

---

## ✨ Experience

- 🌠 680 twinkling stars with multi-layer parallax depth
- 🌫️ Breathing nebula clouds — purple, pink, blue
- 🪐 5 planets — each one a real project
- 🌙 Shooting stars every 25 seconds
- 🖱️ Full mouse parallax — real 3D depth illusion
- 🎬 Cinematic intro sequence on load
- 🔊 Ambient space audio (Web Audio API)
- ⭐ Community star system — anyone can add themselves

## ⭐ Community Stars

Anyone in the world can click **"Add Your Star"** and become part of the universe. Stars are saved permanently in Supabase and visible to every visitor.

---

## 🛠️ Tech Stack

| Tech | Purpose |
|------|---------|
| HTML + CSS + Canvas API | Full universe rendering |
| Vanilla JavaScript | Animations, parallax, interactions |
| Web Audio API | Ambient space sound |
| Supabase | Community star database |
| Vercel | Global deployment |

**Single file. Zero frameworks. Zero dependencies.**

---

## ⚡ Performance

- `requestAnimationFrame` loop — silky smooth
- Canvas batching for 680+ stars
- Tab inactive → everything pauses
- Mobile optimized — touch support
